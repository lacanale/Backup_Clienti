
--- DEUTSCH ---

Mit der Tachometer- Version (SW Amplifier) 1.24 wurde eine Funktionserweiterung realisiert.
-------------------------------------------------------------------------------------------
Tachometer mit Firmware Version 1.24 und höher müssen daher mit der Konfigurationssoftware 
Version 1.15 und höher konfiguriert werden.
Tachometer mit älterer Firmware können auch mit der neuen Konfigurationssoftware 
konfiguriert werden. Entsprechende Fehlermeldungen weisen dabei auf die nicht existierende 
Funktion hin.
Um diese Fehlermeldungen zu umgehen, kann auch eine ältere Konfigurationssoftware verwendet 
werden.



--- ENGLISH --- 

With the tachometer software release  (SW Amplifier) 1.24 comes a new function.
-------------------------------------------------------------------------------
Tachometer with the firmware version 1.24 and higher have to be configured by the 
configuration software version 1.15 or higher.
Tachometer with older firmware can also be configured by the new configuration software.
Error messages will inform about the not supported function.
To avoid these messages you can also use the old configuration software.



--- FRANCAIS --- 

Une fonction additionnelle pour les versions de tachymètres (Amplificateur SW) 1.24
-----------------------------------------------------------------------------------
Les tachymètres avec un firmware version 1.24 ou supérieur devront être configurés avec un 
logiciel de configuration version 1.15 ou supérieure.
Les tachymètres avec un firmware plus ancien peuvent aussi être configurés avec ce même 
logiciel. Les fonction non-existante seront signalées par des messages d'erreurs 
correspondants.
Une version précédente du logiciel de configuration peut-être utilisée afin d'éviter 
ces messages d'erreur.
